## 📌 Project Overview

Fundraising campaigns generate massive amounts of data, but turning that data into meaningful business decisions can be challenging.

**UnityRaise** is a Business Intelligence project that transforms fundraising data into interactive visualizations, helping organizations understand campaign performance, donor behavior, marketing effectiveness, and revenue generation.

The dashboard provides decision-makers with a complete overview of fundraising activities through dynamic reports and KPI monitoring.

---

## 🎯 Business Problem

Organizations often struggle to answer questions like:

- Which fundraising campaign performs best?
- Where should marketing budgets be invested?
- Which fundraising method generates the highest return?
- Which categories need improvement?
- How can donor engagement be increased?

This dashboard answers these questions through data-driven insights.

---

## 🚀 Solution

Using **Microsoft Excel** for data preparation and **Google Looker Studio** for visualization, this project converts raw fundraising data into an interactive dashboard that enables organizations to:

- Monitor fundraising performance
- Track donation trends
- Compare fundraising methods
- Evaluate marketing effectiveness
- Improve strategic decision-making
